Background

Existing Method

Before Fred came to save the day, employees manually inserted plastic tabs into each desired plug slot, breaking them off from a molded hexagonal connector. This process was time consuming, boring, repetitive, straining on the hands, and prone to human error.

Value Proposition

We replace the manual process with an automated system that automates a boring and repetitive task, prevents strain on employee hands, removes the need for molded parts, and eliminates human error. We do this with the help of Fred, a 6-axis robot that dispenses and cures resin in the desired plug slots. We also provide a web app that allows nontechnical users to control and update the process.

The KeyAssembler

Process

  1. Employee loads plug trays with new plugs.
  2. Employee creates a plug config on the web app.
  3. Employee starts a new plug job on the web app.
  4. Web app publishes the job to an application programmable interface (API).
  5. Robot gets information about the job by visiting the API using the Internet.
  6. Robot dispenses resin for the desired plug configuration.
  7. Robot uses UV lights to cure the resin.
  8. Robot sends a message to the web app API telling it the job status.
  9. Employee sees the job is done and replaces the plugs with a new batch.

Web App Component

The web app component is what you are looking at right now. It is a web app that allows nontechnical users to create plug configurations and start plug jobs. It also allows users to see status updates, read documentation, view insights, and more.

Robotics Component

The robotics component consists of a 6-axis robot that dispenses and cures resin in the desired plug slots, base plate aligner that ensures consistent placement, and plug tray that holds plugs. It is controlled by the web app component.
